### @veritem/eslint-config

![npm](https://img.shields.io/npm/v/@veritem/eslint-config)

```bash
pnpm install -D @veritem/eslint-config
```

#### Config `.eslintrc.json`

> If you're building a frontend app

```json
{
  "extends": "@veritem"
}
```

> If you want to use it in node 


```json
{
    "extends": [
        "@veritem/base",
        "@veritem/ts"
    ]
}
```
